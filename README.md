# GuiaGenshin
Este projeto tem como objetivo ajudar jogadores de nivel iniciante que enfrentam dificuldades ao começar o jogo, explicando desde a gameplay básica até nível avançado, envolvendo interações com elementos e personagens, ensinando a como montar times eficientes e com boas reações elementares. 

## Sobre o projeto
Este projeto foi desenvolvido com o objetivo de ensinar conceitos básicos do jogo Genshin Impact de forma interativa, utilizando HTML, CSS e JavaScript.

- O guia contém:

* Sistema de navegação por abas
* Explicação dos elementos
* Sistema de reações elementais
* Guia de equipe
* Quiz interativo

## Versão
1.0.0

## CI 
O projeto possui integração com GitHub Actions, que executa automaticamente os testes a cada push.

- Arquivo responsável:

.github/workflows/ci.yml

## Problema
Iniciantes em Genshin Impact têm dificuldade em entender mecânicas básicas do jogo.

## Solução
Aplicação interativa com guia, explicações e quiz para aprendizado.

## Público-alvo
Jogadores iniciantes.

## Como rodar
- Clonar o repositório
- git clone https://github.com/seu-usuario/seu-repo.git
- Instalar dependências
- npm install
- Rodar os testes
- npm test
- Abrir o projeto

Basta abrir o arquivo index.html no navegador.

## Testes
O projeto utiliza o Jest para testes automatizados.

Testes implementados:
- Navegação entre abas (goTab)
- Exibição de reações (showReaction)
- Sistema de respostas do quiz (answer)
